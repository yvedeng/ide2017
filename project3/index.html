Project 3

Part 1: Family Tree

We take the graph from google for each member of Trump family and set this graph ase our shape of node. For the tree layout, we show the relationship between parents and children. For each node, when moving mouse over this node, it shows the name and parents of the character denoted by this node. And if we click this node and if it has children nodes, these children nodes will be folded. If click again, it will be restored.

Part 2: Explanations

  1. Motivation

    2. We choose tree layout simply because we were asked to create a family tree. The reason we did not show the partner information is that we do not want to mess up the relationship between parents and children as same as the tree structure. We have imagine that if we add the partners part into our visualization : We set Donald as root and we have to set Donald's partners as the children of this root. This will cause a problem for his children who have a partner, and his grandchildren node will be the children of his son/daughter in law's node. This is a mess. Therefore to make everything clear, we decided to make it this way.

    The info box when the cursor move over the picture was created since we want to show as much as information as possible.

    We decide to use their picture to illustrate their relationship because it looks nicer and friendlier comparing to the text.
