Project 3

Part 1: Family Tree

We took photo from google for each member of Trump family and set this photo as our shape of node. For the tree layout, we show the relationship between parent and children. When mouse is hovering over the node, it shows the name, both parents, current and previous spouses of the person denoted. Nodes can be clicked, if it has children nodes, these children nodes will be folded or unfolded.

Part 2: Explanations

  1. Motivation

    2. We choose tree layout simply because we were asked to create a family tree. The reason we did not show the partner information is that we weren't able to successfully code it and we didn't want to mess up the relationship between parents and children using same branches for both or do it manually (hardcoding nodes based on our data). Therefore to make everything clear, we decided to make it simplier, following only Trump family tree and adding spouses to information box which is shown when hovering over the persons photo.

    The info box when the cursor move over the picture was created since we want to show as much as information as possible.

    We decided to use their picture to illustrate their relationship because it looks nicer and friendlier comparing to the text. As well as almost everyone have seen some pictures of Donald Trump children but very few know their names. That helps to relate to these people.
